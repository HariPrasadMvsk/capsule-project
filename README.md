# React-capsule-project
Chat Application with ReactJS

### Dependencies & DevDependencies -
```
Express, react, react-dom, react-scripts & concurrently
```

### To install all the dependencies -
```
cd capsule-project
npm install
```

### To Run the Local Server -
```
npm start
```
